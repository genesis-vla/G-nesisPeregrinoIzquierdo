ESTE ES UN EJEMPLO PARA MODIFICAR EL ARCHIVO

Pasos para subir un archivo a GitHub

  1. Entrar a la página oficial de GitHub: https://github.com
  2. Iniciar sesión con tu usuario y contraseña.
  3. Hacer clic en el botón New para crear un nuevo repositorio.
  4. Escribir el nombre del repositorio y presionar Create repository.
  5. Dentro del repositorio, hacer clic en Add file.
  6. Seleccionar Upload files.
  7. Arrastrar el archivo o seleccionarlo desde tu computadora.
  8. Escribir un mensaje en Commit changes.
  9. Presionar el botón Commit changes para finalizar.
